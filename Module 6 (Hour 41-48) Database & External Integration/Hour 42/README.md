# Hour 42 - PostgreSQL CRUD

This project demonstrates basic CRUD operations with PostgreSQL using `database/sql`.

## Concepts Covered

- create table
- insert row
- select rows
- update row
- delete row

## Run

```powershell
go mod tidy
.\run.ps1
```

## Table Used

The example creates and uses the `hour42_users` table inside the configured PostgreSQL database.

## Requirement

PostgreSQL must be running before you start this example.

If you use Docker for PostgreSQL, start Docker Desktop first, then start the container before running `.\run.ps1`.
